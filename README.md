# id_copy
A simple duplex ID card photocopy tool for Windows and Linux. Currently only works on scanners with duplex scanning and the standard CR80 ID card size (85.6mm x 54mm).

## Requirements
### Linux
 - **NAPS2:**
    - Download: [naps2.com](https://www.naps2.com) or through your distro's package manager.
    - Source: [Github](https://github.com/cyanfish/naps2)
